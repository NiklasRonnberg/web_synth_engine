Web Synth Engine

This is a game synth engine with four voices, inspired by the sounds of my childhood computers and video games. I created it to explore the Web Audio API and experiment with its possibilities. The voices are:

  • 1 - Pulse wave with pulse width adjustment (MIDI channel 1)
  • 2 - Pulse wave with pulse width adjustment (MIDI channel 2)
  • 3 - Triangle wave (MIDI channel 3)
  • 4 - Noise with adjustable clock/quality (MIDI channel 4)

All voices respond to MIDI note number, Pitch Bend, and the following settings:

  • Attack (MIDI CC 73), Decay (MIDI CC 75), and Release (MIDI CC 72) — all voices
  • Pitch bend range (MIDI CC 9) — all voices
  • Portamento time (MIDI CC 5) — voices 1 to 3
  • Vibrato rate (MIDI CC 76) and Vibrato depth (MIDI CC 77) — voices 1 to 3
  • Pulse width (MIDI CC 71) — voices 1 and 2

MIDI CC Mapping Details

  • CC 5 – Portamento Time Width (Voices 1-3):
    MIDI value 0 = no portamento
    MIDI value 127 = long portamento (2 seconds)
    Linear mapping between these values
  • CC 9 – Pitch Bend Range (All voices):
    MIDI value 0 = 2 semitones
    MIDI value 127 = 24 semitones (i.e., 2 octaves)
    Linear mapping between these values
  • CC 71 – Pulse Width (Voices 1 & 2 only):
    MIDI value 0 = 0.5 (square wave)
    MIDI value 127 = 0.02 (narrow pulse wave)
    Linear mapping between these values
  • CC 73 – Attack:
    MIDI value 0 = instant (0 seconds)
    MIDI values 1–127 = exponential mapping from 0.01 to 4.0 seconds
  • CC 75 – Decay:
    MIDI value 0 = no decay (sustain until note off)
    MIDI values 1–127 = exponential mapping from 0.01 to 4.0 seconds
  • CC 72 – Release:
    MIDI values 0–127 = exponential mapping from 0.01 to 4.0 seconds
  • CC 76 – Vibrato Rate (Voices 1–3):
    MIDI values 0–127 = linear mapping from 0.1 Hz to 20 Hz
  • CC 77 – Vibrato Depth (Voices 1–3):
    MIDI value 0 = 0 cents (no vibrato)
    MIDI value 127 = 600 cents (±0.5 octave)
    Linear mapping across the range
