Web Synth Engine

This is a game synth engine with four voices, inspired by the sounds of my childhood computers and video games. I created it to explore the Web Audio API and experiment with its possibilities. The voices are:

  • 1 - Pulse wave with pulse width adjustment
  • 2 - Pulse wave with pulse width adjustment
  • 3 - Triangle wave
  • 4 - Noise with adjustable clock/quality

All voices respond to MIDI note number and the following settings:

  • Attack (MIDI CC 73), Decay (MIDI CC 75), and Release (MIDI CC 72) — all voices
  • Vibrato rate (MIDI CC 76) and Vibrato depth (MIDI CC 77) — voices 1 to 3
  • Pulse width (MIDI CC 71) — voices 1 and 2

MIDI CC Mapping Details

  • CC 73 – Attack:
    MIDI value 0 = instant (0 seconds)
    MIDI values 1–127 = exponential mapping from 0.01 to 2.0 seconds
  • CC 75 – Decay:
    MIDI value 0 = no decay (sustain until note off)
    MIDI values 1–127 = exponential mapping from 0.001 to 1.0 seconds
  • CC 72 – Release:
    MIDI values 0–127 = exponential mapping from 0.01 to 2.0 seconds
  • CC 71 – Pulse Width (Voices 1 & 2 only):
    MIDI value 0 = 0.5 (square wave)
    MIDI value 127 = 0.02 (very narrow pulse)
    Linear mapping between these values
  • CC 76 – Vibrato Rate (Voices 1–3):
    MIDI values 0–127 = linear mapping from 0.1 Hz to 20 Hz
  • CC 77 – Vibrato Depth (Voices 1–3):
    MIDI value 0 = 0 cents (no vibrato)
    MIDI value 127 = 1200 cents (±1 octave)
    Linear mapping across the range
